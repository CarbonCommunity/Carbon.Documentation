# OnChickenScared
Called when a chicken (or similar animal) becomes scared and flees.
### Return
Returning a non-null value cancels default behavior.

### Usage
::: code-group
```csharp [Example]
private void OnChickenScared()
{
	Puts("OnChickenScared has been fired!");
}
```
:::
